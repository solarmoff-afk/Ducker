# Ducker
Простая графическая библиотека в 1 файле (DuckerNative.cpp)

# Зависимости
- GLAD
- Stb_TrueType
- Stb_Image
- FastVector

# Возможности
- Создание прямоугольника, прямоугольника с углами и круга
- Наложение текстуры
- Прямая и кривая линия
- Блюр и тени
- Тексты с поддержкой .otf и .ttf шрифтов

# Поддерживает
- Windows
- Linux
- Android
